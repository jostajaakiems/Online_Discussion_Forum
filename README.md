# Online Discussion Forum

This is a simple **Online Discussion Forum** built using **PHP and MySQL**.  
The system allows users to register, log in, create posts, view posts, and comment on discussions.

## Features
- User registration and login
- Session-based authentication
- Create new discussion posts
- View all posts
- View single post details
- Add and view comments on posts
- Simple and clean user interface

## Technologies Used
- PHP (Core PHP)
- MySQL
- HTML
- CSS
- Apache (XAMPP)

## How to Run the Project
1. Install **XAMPP**
2. Copy the project folder into:
C:\xampp\htdocs\
3. Start **Apache** and **MySQL** from XAMPP Control Panel
4. Open **phpMyAdmin**
5. Create a database (e.g. `online_discussion_forum`)
6. Import the provided SQL file or create tables manually
7. Update database credentials in `db.php`
8. Open browser and visit:
http://localhost/Online-discussion-forum/
